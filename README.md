# BlogLite

A lightweight, modern Full Site Editing (FSE) WordPress blog theme focused on readability, typography, performance, accessibility and SEO-friendly content presentation.

## Features

- Full Site Editing (FSE)
- theme.json support
- Block Patterns
- Block Styles
- Reading progress bar
- Estimated reading time
- Word count
- Dark & Sepia reading modes
- WooCommerce ready
- RTL support
- Accessibility ready
- Translation ready
- No jQuery
- Vanilla JavaScript
- SOLID architecture
- PSR-4 autoloading

## Requirements

- WordPress 6.8+
- PHP 8.0+
