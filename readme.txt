A lightweight, modern, Full Site Editing WordPress blog theme focused on readability, typography, performance, accessibility, and SEO-friendly content presentation.

== Description ==

BlogLite is a production-ready Full Site Editing (block) theme built for bloggers
who want a clean, fast and readable publishing experience.

Highlights:

* Full Site Editing / block theme powered by theme.json.
* Object-oriented, SOLID architecture with PSR-4 autoloading and dependency injection.
* Zero jQuery. Progressive-enhancement vanilla JavaScript only.
* Reading experience: progress bar, estimated reading time, word count, dark/sepia modes, print styles.
* Accessibility first (WCAG 2.2 AA): skip links, focus styles, ARIA, keyboard navigation, prefers-reduced-motion.
* Performance focused: conditional asset loading, minimal CSS/JS, no render-blocking third-party requests.
* RTL ready and translation ready (full text domain coverage + POT file).
* WooCommerce ready with native, unobtrusive styling.
* Five style variations: Default, Dark, Sepia, Minimal and Magazine.
* Nine professional, responsive and accessible block patterns.

== Installation ==

1. In your WordPress dashboard go to Appearance > Themes > Add New > Upload Theme.
2. Choose the BlogLite zip archive and click Install Now.
3. Click Activate.
4. Open Appearance > Editor to customise templates, parts and styles.

== Frequently Asked Questions ==

= Does BlogLite require any plugins? =

No. BlogLite works out of the box. WooCommerce is supported but optional.

= Is BlogLite translation ready? =

Yes. Every user-facing string is translatable and a bloglite.pot file is
included under /languages.

= Does the theme work with right-to-left languages? =

Yes. BlogLite is fully RTL ready.

= Does BlogLite include a reading progress bar? =

Yes. A fixed progress bar appears on singular posts and pages. It respects
prefers-reduced-motion and degrades gracefully when JavaScript is disabled.
